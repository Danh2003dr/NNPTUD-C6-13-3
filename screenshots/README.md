# Ảnh chụp màn hình Postman (nộp bài)

Đặt 2 file ảnh vào đây:

- **login.png** – Chụp màn hình Postman khi gọi `POST /api/v1/auth/login` và nhận được token.
- **me.png** – Chụp màn hình Postman khi gọi `GET /api/v1/auth/me` với header `Authorization: Bearer <token>` và nhận được thông tin user.

Chi tiết cách gọi API xem file `POSTMAN.md` ở thư mục gốc.
